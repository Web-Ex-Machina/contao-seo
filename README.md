Extension "SEO" for Contao Open Source CMS
======================

The purpose of this extension is to allow agencies, developers and webmasters to extend the SEO management of Contao Websites.

Functionnalities
-------------------
 * Add meta fields to Contao contents 

System requirements
-------------------

 * Contao 3.*
 * Contao 4.* (Manager Edition)


Installation
------------

Clone the extension from Packagist (Contao Manager)
or
Download and paste the folder in system/modules of your Contao Website.
